About Us

We build small websites and demos to teach web fundamentals. This page shows how to create separate HTML files and link them with normal anchor (<a>) elements.

How navigation works

Clicking an anchor with an href like about.html tells the browser to load that file from the same folder. For local testing, save all files in one folder and open index.html with your browser.
